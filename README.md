# SecureX 일일 업무보고서 자동 생성기

SecureX 기술지원팀의 일일 업무보고서 초안을 만드는 실습용 웹앱입니다.

## 포함 파일

- `index.html`
- `style.css`
- `app.js`
- `README.md`
- `messenger_chat_sample.txt`
- `jira_task_sample.csv`

## 기능

- 작성일 입력
- 메신저 대화 내용 입력
- JIRA 업무 데이터 입력
- 샘플 데이터 불러오기
- 보고서 생성
- 보고서 복사
- 초기화

## 실행 방법

1. `index.html`을 브라우저에서 엽니다.
2. `샘플 데이터 불러오기`를 눌러 참고 데이터를 넣습니다.
3. `보고서 생성`을 눌러 초안을 만듭니다.
4. `보고서 복사`로 결과를 복사합니다.

## 참고

- 이 단계에서는 OpenAI API 연결을 포함하지 않았습니다.
- 화면과 기본 동작만 구현된 상태입니다.
- 샘플 파일은 같은 폴더에 그대로 두고 사용합니다.
